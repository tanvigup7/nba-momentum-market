# CourtMomentum

**CourtMomentum** is a simulated basketball momentum prediction market.

## What it does
- Predicts “momentum flips” in basketball games using ML.
- Provides token prices based on predicted momentum probability.
- Trades are simulated with virtual credits (no real money involved).
- Features SHAP explanations for model predictions.
